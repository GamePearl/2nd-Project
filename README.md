# GamePearl
- 자신이 원하는 게임을 찾아보고 그 게임에 대한 여러가지정보(가격,평점 등)을 확인할 수 있습니다.
- 몇가지 질문을 통하여 자신에게 맞는 게임을 추천받을 수 있습니다.
- 미니게임을 통하여 게시판 권한을 레벨업할 수 있습니다.
